# 🛠️ Sistema de Gestão para Assistência Técnica

Um sistema desenvolvido em **Python** para auxiliar assistências técnicas no gerenciamento de clientes, ordens de serviço, estoque e controle financeiro, proporcionando maior organização, agilidade e eficiência no atendimento.

---

## 📌 Funcionalidades

### 👥 Cadastro de Clientes
- Cadastro completo de clientes;
- Histórico de atendimentos;
- Pesquisa rápida.

### 📋 Ordem de Serviço
- Criação automática de OS;
- Controle de status:
  - Recebido
  - Em análise
  - Aguardando peça
  - Em manutenção
  - Pronto
  - Entregue
- Registro de diagnóstico;
- Registro do serviço realizado;
- Valor do serviço;
- Data de entrada e entrega.

### 📦 Controle de Estoque
- Cadastro de peças;
- Entrada e saída de produtos;
- Controle de quantidade;
- Alerta de estoque baixo.

### 💰 Controle Financeiro
- Registro de receitas;
- Registro de despesas;
- Relatórios financeiros;
- Lucro por período.

### 📊 Dashboard
- Total de ordens de serviço;
- Serviços em andamento;
- Serviços concluídos;
- Faturamento;
- Indicadores do sistema.

# 🎯 Objetivo

O objetivo deste projeto é informatizar o gerenciamento de uma assistência técnica, eliminando controles manuais e melhorando:

- Organização
- Produtividade
- Atendimento ao cliente
- Controle financeiro
- Controle de estoque
- Emissão de Ordens de Serviço
